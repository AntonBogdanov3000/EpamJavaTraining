АВТОСЕРВИС
----------------------------------------------
КЛИЕНТ создает учетную запись (марка авто. пробег.год выпуска. номер телефона. почта),заполняет ЗАКАЗ на запись по работам.

ЗАКАЗ содержит данные клиента,наименование работ,дату подачи,
текущий пробег авто,ориентировочную стоимость.ориентировочное время выполнения

АДМИНИСТРАТОР управляет КЛИЕНТАМИ и ЗАКАЗАМИ,
осуществляет прием авто в работу(выбирает дату, определяет стоимость и время выполнения работ)

НЕЗАРЕГИСТРИРОВАННЫЙ ПОЛЬЗОВАТЕЛЬ может просматривать информацию о автосервисе,заказывать обратный звонок.
